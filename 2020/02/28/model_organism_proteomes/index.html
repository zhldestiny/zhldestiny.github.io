2.homo sapiens

1.human proteomes download

Uniprot Proteomes 搜索条件：homo sapiens AND reference:yes

搜索结果如下：

下载第一个protein count有74811的UP000005640, Organism ID:9606

点击Protein Count：74811，下载fasta序列；

2.Uniprot注释下载

限定物种和有结构域注释进行搜索，text格式下载所有

3.从有注释的序列中提取出人类蛋白质组的序列

格式如下：

entry domain evidence length
Q8N8B7 [‘5-82’, ‘173-289’] [‘0000255’, ‘0000255’] 351
O75494 [‘10-88’] [‘0000255’] 262

2.序列处理

​ 2.1split

把下载下来的蛋白质组按照id拆分成一条一条的fasta序列

​ 2.2filter

统计序列长度如下：

num(l<80) num(80=<l<2000) num(l>2000)
11951 61187 789

保留长度在80-2000的序列61187条

​ 2.3处理含有X字符的序列

I-Tasser不能处理含有X的蛋白序列，会报错，删除序列中的X字符

​ 2.4任务提交

从序列列表里每次读区1000条序列，然后分配好提交到节点进行计算

3.结构域预测

​ 3.1使用I-TASSER得到init file

​ 3.2用inint file跑ThreaDom

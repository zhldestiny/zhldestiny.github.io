intro

构建数据库的几个步骤：designing a data model, choosing a database management system (DBMS), implementing your data model, and developing a user-friendly frontend to your database.

13.1 Types of Databases

1.Flat file databases:相当于一个文件夹，分门别类整理了不同文件；

2.relational databases：关系型数据库，使用表格（tables）组织数据，不同表格构成关系网络组成数据库称做database schema.

3.Object-Oriented Databases ：面向对象数据库.

13.2 Database Software

1.Sequence Retrieval System (SRA):SRS was developed at EMBL specifically for use in molecular biology database applications;

2.Oracle

3.MySQL :it’s considered suitable for small and medium-sized database applications.

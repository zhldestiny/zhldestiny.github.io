Fast and accurate short read alignment with Burrows–Wheeler transform

Li H, Durbin R. Fast and accurate short read alignment with Burrows–Wheeler transform[J]. bioinformatics, 2009, 25(14): 1754-1760.

Abstract

基于Burrow-Wheeler Transform（BWT）的backward search开发了Burrow-Wheeler Alignment tool(BWA)，来将测序序列比对到参考基因组上.

Methods

  1. Burrows-Wheeler Transform

    1. 以字符串googol为例，\$为字符结尾标识符，有字符串X=googol$，且X[0]=g,X[1]=o…X[6]=\$

    2. 字符串X每次从右往左移动一位字符，移动l（ l=len(X) ）次，得到后缀字符串组的前身（suffix arrary）[0, 1, 2, 3, 4, 5, 6]

    3. suffix array的前身依据字符串第一个字母在字母表里的顺序排序( \$<a<b..<z )，得到后缀字符串[6, 3, 0, 5, 2, 4, 1]，取每个字符串的最后一个字符得到BWT string=lo$oogg

  2. Suffix Array interval index

    假设W是X的子串，则定义：

    ​ $\underline{R}\left ( W \right )= min\left { k:W\; is\; the\, prefix\, of\; X_{S(k)} \right } $ （1）

    ​ $ \bar{R}\left ( W \right )= max\left { k:W\; is\; the\, prefix\, of\; X_{S(k)} \right }$ （2）

    例如字符串“go”的SA interval为[1, 2]，比对的问题就转化为寻找query reads的SA interval

  3. Exact matching:backward search

    Ferragina and Manzini (2000) proved that if W is a substring of X:

    ​ $ \underline{R}\left ( aW \right )=C\left ( a \right )+O\left ( a, \underline{R}\left ( W \right )-1\right )+1$ （3）

    ​ $ \bar{R}\left ( aW \right )=C\left ( a \right )+O\left ( a, \bar{R}\left ( W \right )\right )$ (4)

    其中C(a)是在字符串X[0,n-2]中比a排序还小的字母个数，eg.C(g)=0,C(l)=2,C(o)=3 在字符串googol中

    O(a, i)为字符a在B[0,i]中出现的次数,eg.O(l,3)=1,B[0,3]=lo$o,l出现1次；

ref：1.《Fast and accurate short read alignment with Burrows–Wheeler transform》

进化算法在生物多序列比对中的应用

发表于 2018-10-09

第一章 生物多序列比对

1.2 多序列比对（MSA）问题是生物信息学中一个尚未解决的问题，被列为NP完全的组合优化问题，想要找到复杂性为多项式的精确算法是不可能的。参考文献《Recent progress in multiple sequence alignment : a survey》
多序列比对是一个NP完全问题，解决此问题的传统算法是渐进算法或迭代算法，但随着序列长度和条数的增多，时空复杂性急剧上升，设计一个具有高灵敏性、高精度且低复杂度的算法，成为解决生物MSA的瓶颈问题。
1.3 多序列比对的基本原理

MSA相关概念

1. 相似性、同源性
相似性是指在序列比对过程中用来描述检测序列和目标序列之间相同 DNA碱基或氨基酸残基顺序所占比例的高低；同源性（homology）：是指从一些数据中推断出的两个基因或蛋白质序列具有共同祖先的结论。 2. 直系同源、旁系同源
直系同源（orthology）：不同物种中有相同功能的同源基因； 旁系同源（paralog）：一个物种内的同源基因（功能发生改变）。

序列比对的分类

1. 按比对序列的数量:分为双序列比对和多序列比对。上序列比对是序列分析的基础。 2. 按比对序列的长度：一般规定超过100bp的序列为长序列。长、短序列的应用范围不同。
*3. 按比对序列的范围：分为全局比对和局部比对。全局比对对给定序列的全长进行比较。

多序列比对的打分方法

1. 目标函数：通过目标函数来说明比对结果的好坏，三种主流的比对函数：比对和函数（sum-of-pairs functions）、一致性函数（consensus functions）和树函数。比对和函数（简称SP函数）使用最为普遍。SP函数需要设置的两个重要的参数：替换矩阵和空位罚分。score=sum（residue）-sum（penalty） 2. 替换记分矩阵：记分矩阵主要是记录在做序列比对时两个相对应的残基的相似度。常用替换记分矩阵有：PAM（）和BLOSUM（）。
PAM矩阵（Dayhoff等，1987）得到；
| 序列相似度 | 40% | 50% | 60% |
| ——– |:—-:|:—-:|:—-|
|打分矩阵 | PAM120 | PAM80 |PAM60|
PAM250适合两个序列之间具有14%到27%残基匹配的序列
BLOSUM矩阵（Henikoff 夫妇，1992）：高于或等于80%相同的序列组成的串可用于产生BLOSUM-80矩阵；因此BLOSUM-62用来比较62%相似的序列，BLOSUM80用来比较80%左右的序列。
*3. 空位罚分：有两种罚分方式，一种是线性空位发愤，每个空位罚同样分数；另一种是仿射空位罚分，分为起始空位罚分（gap open penalty）和延伸空位罚分（gap extend penalty），两种分值的大小安排对空位出现的次数及样式有影响，因此对比对结果有影响。

1.4 多序列比对方法

1. 比对方法
手工比对：通过辅助编辑软件（如BioEdit、SeaView、GeneDoc等软件）的不同颜色显示不同残基，靠分析者观察，手工改变比对状态。
计算机程序自动比对：1.同步法；2.渐进法：例如Clustal。 2. 多序列比对算法
-1. 精确比对算法：Needleman-Wunsch（1970）算法用来求解两条蛋白质序列的全局比对问题；Smith-Waterman（1981）双序列比对局部动态规划算法；
-2. 近似比对算法
